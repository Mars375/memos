    **In src/memos/api/auth.py:**

    1. DELETE the `RateLimitEntry` dataclass (lines 12-17, including `count` and `window_start` fields at lines 16-17) and `RateLimiter` dataclass (lines 20-48) entirely. Per SEC-01, the standalone `ratelimit.py` is the sole rate-limiting mechanism.

    2. In `APIKeyManager.__init__` (line 61), DELETE the line `self.rate_limiter = RateLimiter()`. The `APIKeyManager` should have NO rate-limiter attribute.

    3. REPLACE `APIKeyManager.validate` method (lines 87-92) with this simplified version per SEC-11:
    ```python
    def validate(self, key: str) -> bool:
        """Validate an API key using timing-safe comparison."""
        if not self._hashed_keys:
            return True  # No keys configured = auth disabled
        hashed = self._hash_key(key)
        return any(hmac.compare_digest(hashed, stored) for stored in self._hashed_keys)
    ```
    The old expression `hmac.compare_digest(hashed, hashed in self._hashed_keys and hashed or "")` is convoluted. The `any()` pattern is clearer and genuinely timing-safe across all stored keys.

    4. In `create_auth_middleware`, DELETE the entire rate-limiting block (lines 136-149, including the `return response` on line 149):
    ```python
    # DELETE THIS ENTIRE BLOCK:
    #     allowed, headers = key_manager.rate_limiter.check(api_key)
    #     response = await call_next(request)
    #     for k, v in headers.items():
    #         response.headers[k] = v
    #     if not allowed:
    #         from starlette.responses import JSONResponse
    #         return JSONResponse(status_code=429, ...)
    #     return response
    ```
    REPLACE with a simple passthrough after successful auth validation:
    ```python
    return await call_next(request)
    ```

    5. The final middleware function should have this structure:
    - Check skip paths (/, /health, /docs, /openapi.json) -> call_next
    - If auth enabled: check X-API-Key header -> 401 if missing -> 403 if invalid
    - call_next(request)
    - If auth not enabled: call_next(request)
